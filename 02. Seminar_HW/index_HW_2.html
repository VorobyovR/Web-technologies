- Коммерческое использование

Самым популярным языком среди украинских разработчиков остается JavaScript - 18,8%. На втором месте C# у него второй год подряд положительная динамика. Можно предположить, что это благодаря активному росту геймдев-индустрии. Python уже второй год подряд демонстрирует негативную динамику. Java тоже теряет позиции – в этом году она на третьем месте по популярности.

- Какой язык используете для работы сейчас

В общем, популярность большинства языков в этом году выросла. Нехорошая динамика, не считая Java и Python, есть еще только у Ruby, C, Scala, Clojure и C++. Последний, кстати, продолжает стремительно терять популярность. Не в последнюю очередь из-за снижения доли среди новичков.
